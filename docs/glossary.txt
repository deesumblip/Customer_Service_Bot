TITLE: Glossary
TAGS: glossary, terminology, definitions

CALM: Conversational AI with Language Models; Rasa's hybrid architecture.
Flow: Deterministic dialogue steps executed by the assistant.
Pattern: A global conversational handler (e.g., search, chit-chat).
Enterprise Search: Rasa's RAG implementation for knowledge answers.
Retrieval-Augmented Generation (RAG): Answering by retrieving relevant documents and using them to ground an LLM response.
Embedding: Vector representation of text for semantic similarity search.
FAISS: A fast vector index used for nearest-neighbor search on embeddings.
